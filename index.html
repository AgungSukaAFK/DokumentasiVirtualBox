Introduksi

Oracle VM VirtualBox adalah perangkat lunak virtualisasi, yang dapat digunakan untuk mengeksekusi sistem operasi "tambahan" di dalam sistem operasi "utama. Aplikasi ini dirilis perdana pada tahun 2007 dan masih terus dikembangkan hingga saat ini. Dibuatnya website ini bertujuan untuk mmeberikan tata cara/tutorial penggunaan aplikasi VirtualBox, dengan cara mencoba install mesin dengan sistem operasi Windows 10 dan Linux Mint.

Dalam tutorial ini, kita akan melakukan segala proses penginstalan dengan menggunakan sistem operasi Windows 10 Sebagai OS bawaan atau asli. Teman-teman boleh menyesuaikan sendiri untuk penginstalan dengan sistem operasi bawaan yang lain.



Instalasi VirtualBox

Berikut adalah tahapan dalam menginstal aplikasi VirtualBox:

  1. Download file setup VirtualBox, kunjungi halaman download resmi VirtualBox Download

    2. Klik link "Windows hosts" untuk mengunduh file instalasi VBox Windows

  2. Jalankan file instalasi jika sudah terunduh, dan ikuti alur instalasinya sampai selesai

    3. Klik "Next" dan "Install", lalu tunggu hingga proses instalasi selesai. Tekan tombol "Finish" jika sudah selesai

  3. VirtualBox berhasil terinstal!

    4. Aplikasi VBox atau VirtualBox selesai terinstal dan sudah bisa digunakan.



Mendapatkan ISO Linux Mint

Linux Mint merupakan distribusi Linux berbasis Debian-Ubuntu yang bertujuan untuk "membuat sistem operasi yang modern, elegan, dan nyaman yang kuat dan mudah dipakai". Dibuat oleh Clement Lefebvre, dan dikembangkan secara aktif oleh tim maupun komunitas yang ada di dalamnya

Pada tutorial kali ini kita akan memakai Linux Mint 21.2 "Victoria". Kita akan menggunakan varian yang paling populernya yaitu "Cinnamon Edition".
Berikut cara mendapatkan file ISO Linux Mint:

  1. Buka link download ISO Linux Mint 21.2 "Victoria" Cinnamon Edition

  2. Scroll kebawah, dan pilih salah satu server untuk digunakan mengunduh file ISO

    3. Dalam praktik kali ini saya akan menggunakan server Indonesia untuk menguduh file

  3. Klik link untuk mengunduh file ISO


Mendapatkan ISO Microsoft Windows 10

Microsoft Windows adalah sistem operasi komputer pribadi yang dikembangkan oleh Microsoft sebagai bagian dari keluarga sistem operasi Windows NT. Pada praktik kali ini kita akan menggunakan Windows 10, yang diperkenalkan pada tanggal 30 September 2014, dirilis pada 29 Juli 2015 dan pada November 2015, Threshold 2 dari Windows 10 dirilis kepada publik.

  1. Mendapatkan ISO Windows 10

    2. Untuk mendapatkan file ISO pada praktik kali ini kita akan membuatnya dengan Windows Media Creation Tool. Atau bisa juga mendownload langsung melalui website lain seperti Nesabamedia

  2. Buka Media Creation Tool yang sudah didownload

    3. Klik tombol "Accept"

    Pilih opsi "Create installation media", lalu klik tombol "Next"

    Pilih opsi "ISO file", lalu klik tombol "Next". Pilih tempat folder untuk menyimpan file ISO

  3. Tunggu hingga prosesnya selesai.

    4. File ISO sudah terdownload



Menggunakan Linux Mint dengan VirtualBox

Sebelum memulai, pastikan anda sudah menginstall aplikasi VirtualBxx dan file ISO untuk Linux Mint. jika belum, anda bisa mengikuti bagian ini untuk menginstall VirtualBox dan Linux Mint terlebih dahulu.

  1. Buka aplikasi VirtualBox

    2. Buat mesin baru dengan mengklik tombol "new" atau melalui opsi "Machine->New", atau bisa juga dengan menekan shortcut ctrl + N

    Masukkan kolom name dengan nama yang kalian inginkan, disini saya menggunakan nama "LinuxMint". Lalu masukkan file ISO yang sudah kalian buat. Klik "Next"

    Klik "Next"

    Masukkan jumlah RAM dan jumlah CPU yang akan digunakan, contohnya disini saya menggunakan 4 GB RAM dengan 2 CPUs. Sesuaikan dengan spesifikasi mesin yang anda miliki. Lalu klik "Next"

    Masukkan jumlah storage yang akan digunakan, disini saya memberikan storage minimal yaitu sebesar 10 GB. Lalu klik "Next"

  2. Menjalankan Linux Mint

    3. Jika Mint sudah termuat, gunakan tombol panah untuk mengganti opsi dan enter untuk memilih opsi. Pilih opsi "Start Linux Mint ..." untuk melanjutkan proses booting ke Linux Mint

  3. Selesai

    4. Berhasil booting ke Linux Mint dan Virtual Machine sudah bisa digunakan!



Menggunakan Windows 10 dengan VirtualBox

Sebelum memulai, pastikan anda sudah menginstall aplikasi VirtualBxx dan file ISO untuk Windows 10. jika belum, anda bisa mengikuti bagian ini untuk menginstall VirtualBox dan ISO Windows 10 terlebih dahulu.

  1. Buka aplikasi VirtualBox

    2. Buka VirtualBox dan klik "new" untuk membuat mesin virtual baru.

    Isi kolom nama, dan isi kolom ISO Image dengan path file ISO Windows 10 yang sudah disiapkan. Jika belum ada ikuti langkah ini untuk membuat ISO Windows 10

    Klik "Next". Jika ada error pada penamaan "Host Name", ubah namanya dengan tidak menggunakan spasi atau karakter unik.

    Contoh hostname yang salah: Windows 10 - belajar

    Penamaan hostname yang benar: Windwows10_belajar atau Windows10Belajar

    Pilih jumlah RAM dan CPU untuk digunakan. Bergantung pada hardware yang kalian miliki, disarankan untuk tidak memilih jumlah dengan bar berwarna merah. Contohnya disini saya memberikan 4GB RAM dan 2 CPUs.

    Tetapkan jumlah storage untuk digunakan pada mesin virtual, untuk sistem operasi windows 10 dibutuhkan minimal storage sebesar 10GB. Contohnya disini saya memberikan ruangan sebesar 15GB.

    Lalu klik "Next" dan "Finish"

  2. Buka tab baru VirtualBox

    3. Setelah mengklik tombol "finish" maka akan muncul tab baru yang menjalankan sistem operasi Windows 10. Beralihlah ke tab tersebut untuk melanjutkan proses.

  3. Tunggu proses instalasi berjalan

  4. Windows 10 Siap digunakan!

    5. Setelah proses instalasi selesai, kita sudah bisa menggunakan mesin virtual dengan OS Windwows 10 ini.
